# KI Notes

## 2026-04-21 UI simplification

- Strom Ampel should stay a one-glance app, not a dashboard.
- Homepage priority: `Hero -> Reminder -> One timeline area -> Footer`.
- Avoid parallel advisory systems; the Hero card should remain the main decision surface.
- Prefer one timeline with `Today / Tomorrow` tabs over two full cards.
- Keep reminder setup centered on smart reminders; if needed, one-time reminders should stay hidden or secondary.
- Keep settings focused on frequent choices; move technical options into an advanced section.
- Keep first-run privacy copy short: local settings, anonymous analytics, privacy link.
- Default rule for copy: one line if possible, neutral tone, no FOMO.
- Hero card should show the large effective price, one short fee note, and one short next-window hint.
- Tight spacing helps: fewer cards, smaller gaps, less subtitle weight.
